# 🌍 Language Translation Tool

Built as part of **CodeAlpha AI Internship — Task 1**

## 📋 About

This is a simple Language Translation Tool that allows users to enter text, select a source and target language, and get instant translated output.

## 🎯 Features

- Translate text between 10+ languages (English, Urdu, French, German, Spanish, Arabic, Chinese, Japanese, Turkish, etc.)
- Simple and clean user interface built with Gradio
- Real-time translation using Google Translate API (via deep_translator)

## 🛠️ Technologies Used

- **Python**
- **deep_translator** — for handling translation requests via Google Translate API
- **Gradio** — for building the interactive web interface

## 🚀 How to Run

1. Install required libraries:
   pip install -r requirements.txt

2. Run the script:
   python language_translation.py

3. The Gradio interface will open in your browser — enter text, select languages, and translate!
