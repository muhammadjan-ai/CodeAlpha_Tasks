# 🤖 FAQ Chatbot

Built as part of **CodeAlpha AI Internship — Task 2**

## 📋 About

An AI-powered FAQ chatbot that matches user questions with the most relevant FAQ using TF-IDF and Cosine Similarity, built with a Streamlit interface.

## 🎯 Features

- Matches user queries to the closest FAQ using NLP techniques
- Clean, interactive Streamlit UI with chat history
- Fallback response for unmatched questions
- Terminal version also included (chatbot.py)

## 🛠️ Technologies Used

- **Python**
- **Pandas** — for reading FAQ data from Excel
- **Scikit-learn** — TF-IDF Vectorizer & Cosine Similarity for matching
- **Streamlit** — for the interactive web interface

## 🚀 How to Run

1. Install required libraries:
   pip install -r requirements.txt

2. Run the Streamlit app:
   streamlit run app.py

   Or run the terminal version:
   python chatbot.py
